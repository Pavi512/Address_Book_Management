# Address Book Management

A Java console-based application that demonstrates the implementation of an Address Book Management System using Object-Oriented Programming (OOP) concepts. The project is developed incrementally through multiple use cases, including contact creation, contact management, multiple contacts, and multiple address books. Each feature is implemented in a separate Git feature branch and merged into the development branch using a structured Git workflow.

---

## About the Project

The Address Book Management System allows users to create and manage contacts efficiently through a console-based interface. It demonstrates the practical implementation of Java Object-Oriented Programming concepts such as encapsulation, collections, composition, and file organization while following a feature-branch Git workflow.

The application progresses through six use cases, beginning with creating a Contact class and ending with managing multiple Address Books using a dictionary.

## Git Workflow

The project follows a feature branch workflow.

- `main` – Initial repository setup
- `dev` – Development branch
- `feature/UC1-ContactClass` – Contact class implementation
- `feature/UC2-UC3-Implementation` – Add, Edit, Delete contacts
- `feature/Main` – Main application implementation
- `feature/UC4-UC6-UpdatedMain` – Multiple Address Books using Dictionary

Each feature branch represents a separate use case and is merged into the `dev` branch after successful implementation, ensuring a clean and organized Git history.
